
### 代码生成器
##### 功能介绍
1. 生成springboot  +  mybatisplus 增删查改
2. 生成 vue 前端文件  列表、编辑、保存
3. 支持 swagger 
4. 支持 lombok

#### 使用

```xml
    <dependency>
        <groupId>com.muluofeng</groupId>
        <artifactId>generator-boot-stater</artifactId>
        <version>1.0-SNAPSHOT</version>
    </dependency>

```
